This game tracks location for use in-game. This data is not stored or used for any other purposes.

Controls

Up / Down Arrows - Swap blocks

Left / Right Arrows - Shift block grid position

Shift - Drop block

Escape - Pause

All in-game buttons are useable, and do the same as their arrow counterparts, with shift's counterpart being the bottom right arrow button.

The only exception is high scores, which should be typed with keyboard keys and backspace. (Touch feature not fully implemented)
